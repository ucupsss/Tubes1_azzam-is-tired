# Tubes 1: Battlecode 2025 - Azzam is Tired


Cara instalasi dan kompilasi
1.  **Clone repositori yang berisi konfigurasi battlecode ini:**
    ```bash
    git clone https://github.com/Fariz36/STIMA-battle.git
    ```
2. **Clone reporsitori ini yang berisi code algoritma setiap bot**
    ```bash
    git clone https://github.com/ucupsss/Tubes1_azzam-is-tired.git
    ```
3. **Copy folder src ke repo Fariz36**
4.  **Lakukan proses build proyek:**
    ```bash
    ./gradlew build
    ```
5. **Buka folder client lalu run aplikasinya**
