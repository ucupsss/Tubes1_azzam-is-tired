# Tubes 1: Battlecode 2025 - Azzam is Tired

## Penjelasan Singkat Algoritma Greedy Tiap Bot
### 1. Main-bot
Fokus pada pembangunan tower menggunakan soldier dan ekspansi wilayah menggunakan splasher berdasarkan scoring movement tertentu
### 2. Alt-1
Hanya menggunakan robot type soldier untuk bangun tower secara satu per satu dan terkoordinasi dengan message antar robot dan tower
### 3. Alt-2
Mengoptimasi setiap role dari bot dan tower dengan tower untuk keseimbangan spawn dan soldier untuk nge-rush wilayah lawan, dan mopper untuk menghapus cat lawan
## Instalasi 
1. Install JDK dengan versi JDK 21, download dari link di bawah
```bash
https://www.oracle.com/java/technologies/downloads/
```
2. Install gradle, download dari link di bawah 
```bash
https://docs.gradle.org/current/userguide/installation.html
```

## Command atau langkah-langkah dalam meng-compile atau build program
1.  **Clone repositori yang berisi konfigurasi battlecode ini:**
    ```bash
    git clone https://github.com/Fariz36/STIMA-battle.git
    ```
2. **Clone reporsitori ini yang berisi code algoritma setiap bot**
    ```bash
    git clone https://github.com/ucupsss/Tubes1_azzam-is-tired.git
    ```
3. **Copy folder src ke repo Fariz36**
4.  **Lakukan proses build proyek:**
    ```bash
    ./gradlew build
    ```
5. **Buka folder client lalu run aplikasinya**
